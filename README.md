# 🎭 Real-Time Emotion Detection using CNN

A real-time emotion detection system built using **Deep Learning (CNN)** and **OpenCV** that can detect human emotions from facial expressions via webcam.

---

## 🚀 Features

* 🎥 Real-time face detection using webcam
* 😄 Emotion prediction (Happy, Sad, Angry, Neutral, etc.)
* 🧠 Built using Convolutional Neural Network (CNN)
* ⚡ Fast and lightweight model

---

## 🛠️ Tech Stack

* Python 🐍
* TensorFlow / Keras
* OpenCV
* NumPy
